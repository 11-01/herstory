# herstory - 古今中外女性历史网站

这是一个记录、展示古今中外杰出女性人物、历史文章与相关文物的 Flask 网站，旨在通过数字化方式，让更多人了解被历史忽视的女性力量。

---

## 🌟 项目简介
- **人物模块**：收录不同时代、不同领域的杰出女性生平与故事
- **文章模块**：深入解读女性历史事件与文化背景
- **博物馆模块**：展示与女性历史相关的文物、展品信息
- **管理员后台**：支持内容的增删改查，方便维护和更新

---

## 🛠️ 技术栈
- 后端：Python + Flask
- 数据库：SQLite
- 前端：HTML + 原生 JavaScript
- 部署：可本地运行，也可搭配服务部署
